# PhotoEditTools-Amaterasu
PhotoEditTools("Amaterasu") - are applications for image editing: masking, spatial filtering, cubic interpolation, etc.

 ![image](https://user-images.githubusercontent.com/46007591/120107136-7e9e5f80-c168-11eb-88d5-219fe95f892d.png)

Functions:
⦁	Blending images
⦁	Cubic interpolation
⦁	Spatial filtering
⦁	Binarization of images

Blend types:
⦁	Addition (Linear Dodge)
⦁	Substract
⦁	Multiply
⦁	Divide
⦁	Screen
⦁	Difference
⦁	Overlay
⦁	Exclusion
⦁	Soft Light
⦁	Hard Light
⦁	Vivid Light
⦁	Linear Light
⦁	Pin Light
⦁	Hard Mix
⦁	Darken Only
⦁	Lighten Only
⦁	Color Dodge
⦁	Color Burn
⦁	Linear Burn
⦁	Experimental1
⦁	Experimental2
⦁	Experimental3
